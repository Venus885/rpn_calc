# 🧮 RPN Calculator - 逆波兰表示法计算器

一个基于 C++ 实现的高性能逆波兰表示法（RPN）计算器，支持交互式数学表达式计算。

## ✨ 特性

- 🔢 **基本运算**: 支持加、减、乘、除四则运算
- 📊 **栈管理**: 实时查看栈状态，支持清空操作
- 🛡️ **错误处理**: 完善的异常处理（除零、栈空、格式错误）
- 🎯 **高精度**: 支持浮点数运算，6位小数精度
- ⚡ **高性能**: 使用 C++17 标准，优化编译
- 🔧 **易于扩展**: 模块化设计，支持新操作符

## 🚀 快速开始

### 编译项目

```bash
g++ -std=c++17 -Wall -Wextra -O2 -o rpn_calc rpn_calc.cpp
