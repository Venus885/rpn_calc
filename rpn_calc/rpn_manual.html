🚀 项目简介

RPN（逆波兰表示法）是一种数学表达式表示法，其中操作符位于操作数之后。 这种表示法无需括号即可明确运算顺序，被广泛用于计算器和编程语言中。

核心优势：

  • 🎯 无歧义性 - 无需括号，运算顺序明确
  • ⚡ 高效计算 - 适合栈机实现，计算效率高
  • 🔧 易于扩展 - 模块化设计，支持新操作符

✨ 功能特性

🔢 基本运算

支持加、减、乘、除四则运算，处理整数和浮点数

5 3 + → 8.0

📊 栈管理

实时栈状态查看、清空操作，支持连续计算

命令: s (显示栈), c (清空栈)

🛡️ 错误处理

完善的异常处理：除零、栈空、格式错误

10 0 / → 除零错误

🎯 高精度

支持浮点数运算，6位小数精度，科学计算友好

3.14159 2 * → 6.283180

echo "};"
// RPNCalculator 类定义
class RPNCalculator {
private:
    std::stack<double> stack;  // 内部栈存储

public:
    // 栈操作
    void push(double value);           // 压入数值
    double pop();                      // 弹出数值
    
    // 计算功能
    void calculate(const std::string& operation);  // 执行计算
    
    // 工具函数
    void clear();                      // 清空栈
    void displayStack() const;         // 显示栈内容
    bool empty() const;                // 检查栈是否为空
    size_t size() const;               // 获取栈大小
};
